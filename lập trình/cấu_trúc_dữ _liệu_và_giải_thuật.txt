cấu trúc dữ liệu là tập hợp các kiểu dữ liệu được sắp xếp theo 1 trật tự cụ thể
cấu trúc dữ liệu chia làm 2 loại:
	+ cấu trúc dữ liệu tuyến tính
		++ cấu trúc dữ liệu mảng(array)
		++ cấu trúc dữ liệu ngăn xếp(stack)
			+++ sắp xếp theo nguyên tắc LIFO(last in first out)
		++ cấu trúc dữ liệu hàng đợi(queue)
			+++ sắp xếp theo nguyên tắc FIFO(first in first out)
		++ cấu trúc dữ liệu danh sách liên kết(linked-list)
			+++ các phần tử dữ liệu được kết nối thông qua một loạt các nút
				mỗi nút chứa data và địa chỉ nút tiếp theo
	+ cấu trúc dữ liệu phi tuyến tính
		++ cấu trúc dữ liệu đồ thị(graph)
			+++ Spanning Tree and Minimum Spanning Tree
			+++ Strongly Connected Components
			+++ Adjacency Matrix(ma trận kề)
			+++ Adjacency List(danh sách gần kề)
		++ cấu trúc dữ liệu cây(tree)
			+++ Binary Tree
			+++ Binary Search Tree
			+++ AVL Tree
			+++ B-Tree
			+++ B+ Tree
			+++ Red-Black Tree
